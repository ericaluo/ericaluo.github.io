My Secret Diary

An individual developed Android APP as my graduation honour project in 2020. 
Users can record text and unlock the software by setting a password and a particular location.

User Login

For the first time login, the user will be asked to enter a password for further login, however the password cannot be changed after setting up.

Diary Taking

The user is able to edit, change and delete diaries in this APP

Encryption Applied

All information including the password and all the diaries will be encrypted by the AES algorithm and stored in SQLite database.

Location Determination

The APP will ask for GPS permission and check for the user's current location.

Set Up Pass Location

The user is able to set the current location as a secure location. The user can only normally use this APP in the set-up secure location.

Calorie Tracker

A group developed Android APP from Object-oriented Software engineering class, where users can record daily nutrition intake and receive recipe recommendations

User Login

Users are able to log in as student users by registering or Google accounts.

Personal Profile

Users can modify personal profile including weight, height, age and etc. The APP will calculate the needed calorie per day for users.

Recommend Dishes

Users are able to retrieve recommend dishes (Carleton University cafeteria's menu) under the healthy mode

Calorie Calculator

Student users are able to calculate the total calories for selected dishes in self-choosing mode.

Weekly Report

Student users are able to receive a weekly health report in Calorie Evaluation.
